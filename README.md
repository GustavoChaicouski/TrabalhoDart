# Trabalho 1 - Hello World em Dart
##  Autor
Gustavo Santos Chaicouski

##  Objetivo
Este projeto tem como objetivo verificar a correta configuração do ambiente de desenvolvimento para a linguagem Dart, bem como a execução de um programa simples no terminal.

##  Descrição
O programa desenvolvido consiste em um arquivo Dart contendo uma função `main()` que imprime a mensagem **"Hello World"** no terminal.

##  Tecnologias utilizadas
* Dart
* Git
* GitHub

Passos
Clone o repositório:
git clone https://github.com/GustavoChaicouski/TrabalhoDart

Acesse a pasta do projeto:
cd TrabalhoDart

Execute o programa:
dart run main.dart

Saída esperada
Hello World
